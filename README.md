# Qwen3-TTS Colab

Run [Qwen3-TTS](https://github.com/QwenLM/Qwen3-TTS) on Google Colab with a Gradio UI.

## Features

- Text-to-Speech (9 premium voices)
- Voice Cloning (3-sec rapid clone)
- Voice Design (create voices from descriptions)

## Quick Start

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/RedWilly/Qwen3-TTS-Colab/blob/main/Qwen3_TTS_Colab.ipynb)

1. Open the notebook in Colab
2. Select GPU runtime (T4 or better)
3. Run all cells
4. Use the Gradio UI link
